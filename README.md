# LifeOS (Unity Project)

一个基于 Unity 的视觉系统项目，用于探索高自由度的交互体验、粒子效果与沉浸式空间表现。

---

## ✨ 项目特性

- 🎮 Unity 3D 实时渲染
- 🌌 高级视觉效果（粒子 / Shader Graph / 发光系统）
- 📷 相机自由控制系统（Orbit / Zoom / Touch）
- 💫 计划支持粒子沙画 / Gaussian Splat 风格效果
- 🧠 模块化系统设计（便于扩展 UI / 场景 / 特效）

---

## 🧱 技术栈

- Unity 2022+（推荐 URP）
- C# 脚本系统
- Shader Graph
- Git 版本控制
